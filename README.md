# attendance-management
School Attendance Management System

This Python project automates school attendance recording and analysis. Teachers can mark attendance, and admins/students can view attendance trends. Data is stored in MySQL and analyzed using Pandas with visualizations via Matplotlib.

Features

Mark daily attendance
View student-specific or full attendance
Analyze trends (monthly rates, most absent students)
Visualize data in graphs

Dependencies

Python 3.x

MySQL Server

pandas

matplotlib

mysql-connector-python
